# AI Meeting Preparation Agent 📝

An AI-powered meeting preparation agent that generates comprehensive briefings, industry insights, strategies, and executive summaries for any meeting.

## Features
- Company background research
- Industry analysis and trends
- Tailored meeting strategy and agenda
- Executive brief with talking points and Q&A prep

## Requirements
- Python 3.10+
- Gemini API Key (free at https://aistudio.google.com/apikey)
- Serper API Key (free at https://serper.dev)

## Installation

```bash
git clone https://github.com/shobbbhit/ai-meeting-agent.git
cd ai-meeting-agent
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Usage

```bash
streamlit run meeting_agent.py
```

Then enter your API keys in the sidebar and fill in the meeting details.
